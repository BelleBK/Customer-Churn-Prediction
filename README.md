# Customer-Churn-Prediction
This project reproduces a research paper on customer churn prediction using machine learning techniques.

## Dataset
Telco Customer Churn dataset from Kaggle.
Link : https://www.kaggle.com/datasets/blastchar/telco-customer-churn

## Paper work
Churn Prediction using Various Machine Learning Algorithms in Telecom Sector
Link : https://www.ijcaonline.org/archives/volume184/number36/verma-2022-ijca-922462.pdf

## Models Used
- Logistic Regression
- Decision Tree

## Steps
1. Data preprocessing
2. Feature engineering
3. Model training
4. Evaluation

## How to Run
1. Download dataset
2. Open Jupyter Notebook
3. Run all cells

## Results
Logistic Regression performed better than Decision Tree.
